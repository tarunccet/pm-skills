# product-management-skills

This is an alias for [`ai-pm-skills-mcp`](https://www.npmjs.com/package/ai-pm-skills-mcp) — 83 agentic PM skills as an MCP server for Claude, Cursor, Windsurf, and more.

## Usage

Add to your MCP client config:

```json
{
  "mcpServers": {
    "pm-skills": {
      "command": "npx",
      "args": ["-y", "product-management-skills"]
    }
  }
}
```

See [github.com/tarunccet/pm-skills](https://github.com/tarunccet/pm-skills) for the full skill list, setup guide, and contributing guide.
