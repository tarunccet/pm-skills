# 90-Second Demo Walkthrough

Use this as a quick demo script for GitHub, social posts, or a screen recording.

## Goal
Show that the repo helps PMs move from vague prompts to structured workflows quickly.

## Script

### 0–15 seconds
Open the README and say:
> "This is Agentic PM Skills: 9 plugins, 93 skills, and 63 commands for product managers using AI."

### 15–35 seconds
Run:
```text
/discover Improve activation for first-time users
```
Say:
> "Instead of a random brainstorm, it guides discovery with structure."

### 35–55 seconds
Run:
```text
/write-prd Smart notification batching for enterprise teams
```
Say:
> "It moves naturally from discovery into execution artifacts like PRDs."

### 55–75 seconds
Run:
```text
/ai-feature-launch AI assistant for support ticket triage
```
Say:
> "For AI products, it covers model evaluation, risk, metrics, and launch planning."

### 75–90 seconds
Show:
- starter packs
- proof/case studies
- compatibility matrix

Close with:
> "The goal is not more prompting. The goal is better PM decisions with repeatable AI workflows."
