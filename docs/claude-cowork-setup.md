# Using PM Skills with Claude Cowork

> **Best for non-developers** — no terminal or config files required.

## Setup

1. Open **Customize** (bottom-left of the Cowork interface)
2. Go to **Browse plugins** → **Personal** → **+**
3. Select **Add marketplace from GitHub**
4. Enter: `tarunccet/pm-skills`

All 9 plugins install automatically. You get both commands (`/discover`, `/strategy`, etc.) and skills.

## Usage

Once installed, use any command in the Cowork chat:

```
/discover AI-powered meeting summarizer for remote teams
/write-prd SSO support for enterprise customers
/learn — I'm a new PM, what should I learn first?
```

Or just describe your task in natural language — skills activate automatically based on context.

---

## Tips

1. **Start with `/find-skill`** if you're not sure which command to use — it routes you to the right skill.
2. **Chain commands**: After any command, it suggests relevant next steps. Follow the flow!
3. **Use `/learn`** to explore interactive learning modules if you're new to PM or leveling up.
