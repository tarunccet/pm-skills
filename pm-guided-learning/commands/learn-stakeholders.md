# /learn-stakeholders — Stakeholder Management Simulation

Launches the multi-stakeholder simulation where you present Finova's Q3 roadmap to 4 executives with conflicting priorities.

## Invocation

```
/learn-stakeholders               # start the full simulation
/learn-stakeholders communication # focus feedback on executive communication clarity
/learn-stakeholders negotiation   # focus feedback on handling pushback
/learn-stakeholders beginner      # run with 2 stakeholders instead of 4
```

## Workflow

Load and execute the `learn-stakeholder-management` skill. The module will:
1. Brief you on Finova (fintech app) and the Q3 roadmap situation
2. Run Phase 1: Roadmap presentation with stakeholder reactions (Jamie, Morgan, Sam, Riley)
3. Run Phase 2: Escalating pushback and negotiation rounds
4. Run Phase 3: Structured debrief across 5 dimensions (0–20 points each)

## Edge Cases

- If the learner asks to see the stakeholder agendas upfront: decline — discovering stakeholder concerns through the conversation is part of the skill being practiced.
- If the learner makes promises they can't keep (e.g., "we'll do Salesforce integration this quarter"): have the relevant stakeholder immediately ask follow-up questions that reveal the over-commitment; debrief on this explicitly.
- If the learner gets flustered and says something inappropriate: don't end the simulation — let it play out and address it thoroughly in the debrief.
- If the learner wants a harder challenge: add a 5th stakeholder (an enterprise prospect who joins the call unexpectedly).
