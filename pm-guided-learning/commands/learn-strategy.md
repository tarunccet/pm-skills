# /learn-strategy — Product Strategy Module

Launches the interactive learning module on product strategy using Roger Martin's Playing to Win framework.

## Invocation

```
/learn-strategy                   # start the module
/learn-strategy advanced          # start with more challenging prompts
```

## Workflow

Load and execute the `learn-strategy` skill. The module will:
1. Brief you on the MindBridge case study
2. Walk through all 5 cascade choices: Winning Aspiration → Where to Play → How to Win → Capabilities → Management Systems
3. Run the Coherence Test as a live exercise
4. Include competitor reverse-engineering
5. Run quiz checkpoints at each stage

## Edge Cases

- If the learner hasn't heard of Playing to Win: give a 3-sentence overview of the cascade before starting.
- If the learner wants to use a different framework (e.g., OKRs): note that OKRs are execution tools, not strategy tools — proceed with Playing to Win for this module.
- If the learner is a founder rather than a PM: frame each cascade choice as a board-level decision they'd make.
