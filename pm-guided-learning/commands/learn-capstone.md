# /learn-capstone — PM Foundations Course Capstone

Launches the capstone project for the PM Foundations Course — a 6-phase, full-lifecycle product case study where you act as PM for Pando, a B2B collaboration startup.

## Invocation

```
/learn-capstone                         # start the capstone
/learn-capstone resume                  # resume a previous session (describe where you left off)
/learn-capstone "completed modules: 1-8" # start with context about prior modules completed
```

## Workflow

Load and execute the `learn-capstone` skill. The project runs across 6 phases:

1. **Phase 1 — Discovery**: Review 4 user interview excerpts, identify opportunities, build a simplified OST
2. **Phase 2 — Strategy**: Apply the Playing to Win cascade to Pando's growth challenge, run the Coherence Test
3. **Phase 3 — Metrics**: Define a North Star Metric, 3 input metrics, and 2 counter-metrics
4. **Phase 4 — Prioritization**: Apply RICE scoring to 5 initiatives, select and justify top 2
5. **Phase 5 — Execution**: Write a concise PRD for the #1 initiative with success metrics and acceptance criteria
6. **Phase 6 — Stakeholder Presentation**: Present the Q1 plan to simulated leadership, defend against challenges, give a go/no-go recommendation

Each phase ends with a 2-question checkpoint. The session closes with a comprehensive course completion debrief.

## Edge Cases

- If the learner hasn't completed prior modules: acknowledge this, explain which modules the capstone draws on, and let them decide whether to proceed or complete the relevant modules first.
- If the learner wants to skip a phase: let them skip but note that later phases build on the skipped phase's decisions — they'll need to make some assumptions to move forward.
- If the session is interrupted: ask the learner to briefly summarize the decisions made in completed phases, then continue from where they left off.
- If the learner's strategy in Phase 2 is fundamentally flawed: flag it clearly and ask if they want to revise before proceeding — coherence across phases matters more than any single correct answer.
