# /learn-ai-pm — AI Product Management Module

Launches the interactive learning module on building and managing AI-powered product features.

## Invocation

```
/learn-ai-pm                      # start the module
/learn-ai-pm beginner             # start with extra scaffolding and simpler scenarios
/learn-ai-pm advanced             # start with harder trade-offs and edge cases
```

## Workflow

Load and execute the `learn-ai-pm` skill. The module will:
1. Introduce the Assist scenario (AI-powered support feature for a B2B SaaS product)
2. Guide you through 4 stages: Model Evaluation → Responsible AI → Prompt Engineering → AI Metrics & Incident Response
3. Run quiz checkpoints after each stage
4. Provide a final performance summary with recommendations for deeper learning

## Edge Cases

- If the learner already has AI PM experience: start at Stage 3 (prompt engineering) and skip earlier stages.
- If the learner asks for theory before practice: give a 2–3 sentence framing, then start the scenario — hands-on experience builds intuition faster than description.
- If the learner is a developer or ML engineer rather than a PM: frame the module around product judgment (what to build and why) rather than implementation details.
- If the learner wants to apply this to their own AI product: adapt the Assist scenario to match their context before starting.
