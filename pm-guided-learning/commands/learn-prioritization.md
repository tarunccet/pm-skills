# /learn-prioritization — Prioritization Frameworks Module

Launches the interactive prioritization workshop where you apply RICE, ICE, and Opportunity Score to the Notely backlog.

## Invocation

```
/learn-prioritization             # start the module
/learn-prioritization beginner    # start with framework explanations before applying
/learn-prioritization advanced    # jump straight to scoring with no formula reminders
```

## Workflow

Load and execute the `learn-prioritization` skill. The module will:
1. Brief you on Notely (note-taking app) and present the 8-feature backlog
2. Guide you through RICE scoring (with provided estimates)
3. Guide you through ICE scoring (your own judgment)
4. Guide you through Opportunity Score (with provided survey data)
5. Facilitate a synthesis discussion comparing all three framework outputs
6. Run quiz checkpoints between stages

## Edge Cases

- If the learner makes math errors: correct them gently and continue — this is a judgment module, not arithmetic.
- If the learner wants to use a different framework (MoSCoW, Kano): acknowledge it's valid, but complete the three required frameworks first, then compare.
- If the learner asks which framework to use "in real life": answer that question in the synthesis stage — it's a key learning outcome of the module.
