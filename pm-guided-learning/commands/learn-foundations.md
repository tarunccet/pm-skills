# /learn-foundations — PM Foundations Module

Launches the PM Foundations interactive learning module — the starting point for the PM Foundations Course and for anyone new to product management.

## Invocation

```
/learn-foundations                          # start the module
/learn-foundations career-switch            # frame for someone transitioning into PM
/learn-foundations "starting first PM role" # frame for someone who just became a PM
```

## Workflow

Load and execute the `learn-foundations` skill. The module will:
1. Ask two brief context questions (background + motivation)
2. Work through Stage 1: The PM Role in Context (dark mode scenario + PM vs. project manager)
3. Run Quiz Checkpoint 1 (PM role clarity)
4. Work through Stage 2: PM Mindset (3 real scenarios)
5. Run Quiz Checkpoint 2 (PM mindset statements)
6. Work through Stage 3: Framework Overview & Self-Assessment (6 skill areas)
7. Generate a personalized course path recommendation

## Edge Cases

- If the learner already has PM experience: acknowledge this and adjust the difficulty upward — skip the scaffolding and focus on the nuances and edge cases.
- If the learner wants to skip to a specific module: let them know what they'd miss and confirm they want to skip Foundations; if yes, direct them to the specific module command.
- If the learner is from a non-tech background (e.g., a doctor exploring health tech PM): adapt the examples to their industry context.
- If the learner asks "do I need to complete this before other modules?": explain that Foundations provides the conceptual base but isn't strictly required for experienced PMs — it's most valuable for beginners or those wanting to reset their mental model.
